# Your Project

This is a demo project located in the `webapps/demo` directory. It is a web application that includes various HTML pages, servlets, and JavaScript files.

## Software/Applications to be Installed

- Java 8
- Tomcat server 9.34.0

## Project Structure

- **bin/**: Contains scripts and binaries for running the server.
- **conf/**: Configuration files for the server.
- **lib/**: Libraries required by the server.
- **logs/**: Log files generated by the server.
- **temp/**: Temporary files.
- **webapps/**: Contains the web application files.
- **work/**: Working directory for the server.

## Web Application Structure

The web application is located in the `webapps/demo` directory and includes the following files:

- **Header.html**: The header section of the web application.
- **Footer.html**: The footer section of the web application.
- **LeftNavigationBar.html**: The left navigation bar of the web application.
- **Content.html**: The main content of the web application.
- **Login**: The login page.
- **Cart**: The shopping cart page.
- **Home**: The home page.
- **GamesList**: The list of games.
- **ConsoleList**: The list of consoles.
- **TabletList**: The list of tablets.
- **Trending**: The trending items page.
- **autocomplete.js**: JavaScript file for autocomplete functionality (currently missing).

## Steps to Run

1. **Compile the Java Files**: Compile all the `.java` files present in the `WEB-INF/classes` subfolder of the `demo` directory.
    ```sh
    javac webapps/demo/WEB-INF/classes/*.java
    ```

2. **Start the Tomcat Server**: Use the appropriate script from the `bin` directory to start the server. For example, on a Unix-based system, you can use:
    ```sh
    ./bin/startup.sh
    ```
   On a Windows system, you can use:
    ```sh
    bin\startup.bat
    ```

3. **Access the Application**: Open a web browser and navigate to `http://localhost:8080/demo/login.html`.
